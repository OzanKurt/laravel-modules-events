# Upgrade Guide — initial release

The 1.0 line is a greenfield module. There is no previous version to upgrade from.

## Installation

See `README.md`. Key environment knobs:

- `EVENTS_DEFAULT_CURRENCY` — 3-char ISO code; default `USD`.
- `EVENTS_DOCUMENT_DISK` — disk for uploaded documents; default `private`.
- `EVENTS_BROADCAST_CONNECTION` — optional override for broadcast driver.
